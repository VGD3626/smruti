Smruti API Documentation

Integrate Gujarati grammar correction into your apps. This page explains endpoints, request formats, and usage examples.

Base URL

https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space

Authentication

No authentication (beta).

Content Type

application/json

Endpoints

  • POST /correct
  • POST /store-feedback

POST /correct

Beta

Send Gujarati text and receive a grammar-corrected version. Maximum length: 2000 characters.

Request Body

Field Type Required Description
sentence string Yes Gujarati text to correct (≤ 2000 chars).
k1 integer Optional Retriever depth. Default: 5.
k2 integer Optional Refinement depth. Default: 5.

Example request (curl)

curl -X POST \
  https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space/correct \
  -H "Content-Type: application/json" \
  -d '{
    "sentence": "હું ગુજરાતિ ભાષા શીખું છુ.",
    "k1": 5,
    "k2": 5
  }'

Example response

{
  "corrected_text": "હું ગુજરાતી ભાષા શીખું છું."
}

POST /store-feedback

Beta

Submit corrected Gujarati sentences to help improve the model.

Field Type Required Description
incorrect_sentence string Yes The original Gujarati sentence.
correct_sentence string Yes Your corrected version (gold label).

Example request (curl)

curl -X POST \
  https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space/store-feedback \
  -H "Content-Type: application/json" \
  -d '{
    "incorrect_sentence": "હું ગુજરાતિ ભાષા શીખું છુ.",
    "correct_sentence": "હું ગુજરાતી ભાષા શીખું છું."
  }'

Example response

{
  "status": "success",
  "message": "Feedback stored successfully."
}
