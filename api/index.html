Smruti API Documentation

The Smruti API lets you integrate Gujarati grammar correction into your own applications. This page describes the available endpoints, request and response formats, and usage examples.

Base URL

https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space

Authentication

No authentication required (beta). Please use responsibly.

Content Type

application/json

Endpoints

  • POST /correct – Get corrected Gujarati text
  • POST /store-feedback – Submit user feedback / gold corrections

POST /correct

Beta

Sends a Gujarati sentence to Smruti and returns the grammar-corrected version. The input must contain Gujarati text and must not exceed 2000 characters.

Request

POST   /correct

Body (JSON)

Field Type Required Description
sentence string Yes Gujarati text to correct. Must include Gujarati characters and must be ≤ 2000 characters.
k1 integer Optional Internal retrieval parameter (1–20). Default: 5.
k2 integer Optional Internal refinement parameter (1–20). Default: 5.

Validation Rules

  • Input must contain Gujarati characters.
  • Input may not be empty.
  • Input length must be ≤ 2000 characters.
  • Gujarati text must be meaningful (more than 1 non-space character).
  • k1 and k2 must be integers between 1 and 20.

Example request (curl)

curl -X POST \
  https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space/correct \
  -H "Content-Type: application/json" \
  -d '{
    "sentence": "હું ગુજરાતિ ભાષા શીખું છુ.",
    "k1": 5,
    "k2": 5
  }'

Example response

{
  "corrected_text": "હું ગુજરાતી ભાષા શીખું છું."
}

POST /store-feedback

Beta

Stores user-provided corrections. The incorrect_sentence must contain Gujarati text. This feedback is used to improve future versions of the model.

Request

POST   /store-feedback

Body (JSON)

Field Type Required Description
incorrect_sentence string Yes The original (incorrect) Gujarati sentence. Must contain Gujarati text.
correct_sentence string Yes The corrected (gold) version provided by the user.

Example request (curl)

curl -X POST \
  https://vrund1346-Smruti-Gujarati-grammar-checker.hf.space/store-feedback \
  -H "Content-Type: application/json" \
  -d '{
    "incorrect_sentence": "હું ગુજરાતિ ભાષા શીખું છુ.",
    "correct_sentence": "હું ગુજરાતી ભાષા શીખું છું."
  }'

Example response

{
  "status": "success",
  "message": "Feedback stored successfully."
}
