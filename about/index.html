About the Project

Smruti is a research-backed Gujarati Grammatical Error Correction (GEC) system designed to make Gujarati writing clearer, correct, and more natural. It uses modern Large Language Models (LLMs) along with a unique non-parametric memory mechanism, allowing the model to continuously learn from real user corrections and improve over time.

This project is part of a larger research work:
“Smruti: Grammatical Error Correction for Gujarati using LLMs with Non-Parametric Memory” .

The research introduces:

  • A memory-augmented LLM pipeline (M1 and M2 modules)
  • A Gujarati-adapted version of ERRANT for fine-grained error analysis
  • A curated dataset of Gujarati grammatical errors
  • A continuous user-feedback mechanism for model improvement

This work aims to make Gujarati NLP more accessible by providing the first full-stack GEC pipeline for the language- covering dataset creation, error analysis, memory modules, and a user-ready correction system.
